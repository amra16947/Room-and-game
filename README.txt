# Unity and Blender

Scenario za igru Sakupljac smeca-kockica

Cilj igre je sakupljanje kockica koje se pojavljuju po sobi.
Igrac ima 5 života. 
Život gubi ako se susretne sa bombom. Bomba oduzima 1 život.


Lista funkcionalnost:
- postoji meni: Nova igra, Prethodna igra, Spašavanje igre, Highscore, Help i Izlaz
- Spašavanje igre - pamti bodove, poziciju igraca, poziciju objekata
- sve stavke menija rade
- kretanje glavnog igraca
- random premjestanje kocke
- hvatanje kocke i povecavanje bodova
- stvaranje i radnom pomjeranje bombi
- susret bombe i igraca je gubitak zivota
- kulminacija igre, bomba pocinje brze da se krece
- svjetlo se pali gasi na klik-space
- kocke se boje u crvenu, zelenu ili plavu, pritiskom na dugme tastarure C, Z ili P
- model sobe je importovan i uklopljen u unity
